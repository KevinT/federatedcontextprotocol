# Licensing

This repository contains two kinds of material; dual-license accordingly before first public release.

- **Specification text** (`SPEC.md`, `CONFORMANCE.md`, `README.md`, `decisions/`, `schema/`): recommended **CC BY 4.0** - anyone may adopt, implement, and adapt the protocol with attribution.
- **Reference tooling** (`tools/`, `templates/`): recommended **MIT** - permissive reuse in any implementation.

> Action required: drop the full CC BY 4.0 text and MIT license text into `LICENSE-spec` and
> `LICENSE-code` respectively, and add SPDX headers to source files. This file only records the
> recommended choice.
